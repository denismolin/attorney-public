Migration / Sauvegarde

Exporte les vecteurs ChromaDB (déjà calculés), la base SQLite (registre, graphe, enrichissement, évaluations), les pièces jointes et le texte OCR dans une archive .zip. À l'import sur un autre poste, tout est réinjecté sans repayer l'indexation. Les clés API ne sont pas incluses.

⚠️ Avant d'importer : configurez le même modèle d'embedding que celui de l'archive (menu Réglages).
